## Installation

- Go to [Releases](https://github.com/3Dvachevsky/Layers/releases) and download the latest version.
- Add the downloaded folder `ez_change_props` to the addons
- `Project -> Project Settings -> Plugin -> Layers -> Status -> Enable`
